Recursion...Recursion...Recursion...

10/19/2014

Alright, folks--time to talk about something that you might never need to know--but you'll be glad you've heard of it if you do--Recursion! Recursion is not common in Ruby, Python, C, C++, or Java, but it's still an interesting topic.

The basic idea of recursion is to take a problem with multiple steps--the example I've seen used multiple times is factorials--and break that down into smaller steps. You create a method that calls itself after running so that it will do a very simple task multiple times, then give you the final output.


def recursive_factorial(n)
 if n == 0
 return 1
 else
 n * factorial(n-1)
 end
 end

This is what a recursive solution for factorials looks like in Ruby. Kind of cool--but not as sexy as this...


def iterative_factorial(n)
 (1..n).inject{|answer, n| answer * n}
 end

Isn't that nice? Fast and simple.

And that's the reason languages like Ruby don't typically bother with recursion--we have iteration at our hands!

Also--super nerdy joke--do a quick Google search for "Recursion". It took me a second to get the joke--but, hey--maybe you're smarter than me. Either way, I got a good chuckle out of it.
